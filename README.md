## What is SnapLens?

People take screenshots of assignments, payments, notes, and links â€” but they get buried and forgotten. SnapLens uses AI to:

- **LLM:** Gemini / OpenAI API
- **Database:** SQLite (MVP)
- **Frontend:** HTML + JS + Tailwind CSS
- **Extension:** Chrome Manifest v3
